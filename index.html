Welcome to bouledogue

Bouledogue is a simple UI framework built on top of Skeleton and Systematize currently in developpement for learning purposes.


GitHub : Pefouk/Bouledogue

NPM : bouledogue

It features :
  • Grid system (from Skeleton)
  • Animations and timing animations
  • Buttons
  • Basic page layout (Header, Content, Footer)
  • Colour scheme customisation

Because it is including Systematize, pages made with Bouledogue will look the same on every navigator.

It is under heavy development at the moment, huge changes can happen anytime so it is not suited for production environment, yet.
